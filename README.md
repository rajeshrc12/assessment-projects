# Project Setup

## Installation

### Client
1. Copy `.env.example` → `.env`
2. `cd client`
3. `npm install`
4. `npm run dev`

### Server
1. Copy `.env.example` → `.env`
2. `cd server`
3. `npm install`
4. `npx prisma migrate dev`
5. `npx prisma generate`
6. `npm run dev`
